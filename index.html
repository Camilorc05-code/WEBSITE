⭐ Featured · 2026 · Personal Project
TaskFlow Backend API

A professional-grade task management REST API built from scratch with FastAPI. Full auth flow, team management, paginated CRUD — containerized and tested in production.

  • JWT Authentication — secure register, login, protected routes
  • Password hashing with bcrypt via Passlib
  • Teams & Tasks with ownership validation
  • Pagination and filtering on list endpoints
  • PostgreSQL + SQLAlchemy ORM + Alembic migrations
  • Dockerized with Docker Compose
  • Automated tests with Pytest (3 passed ✓)
  • Full API docs via Swagger / Redoc
🐍Python ⚡FastAPI 🐘PostgreSQL 🗃️SQLAlchemy 🐳Docker 🔐JWT 🧪Pytest
